CSS nima? 
 
   Cascading uslublar jadvali (CSS) veb sahifa tartibini formatlash uchun ishlatiladi. CSS yordamida siz 
                    rangni,shriftni,matn hajmini,elementlar orasidagi masofani,elementlar qanday joylashishini va qanday  fon
                    tasvirlari yoki fon ranglarini ishlatishni, turli qurilmalar va ekran o`lchamlari uchun turli displeylarni
                    boshqarishingiz mumkin.Shu va shunga o`xshash funksiyalar CSS da amalga oshiriladi!       
                            
CSS-dan foydalanish 

                    CSS-ni HTML hujjatlariga 3 usulda qo`shish mumkin:
                    • Inline  style - HTML elementlari ichidagi atribut yordamida 
                    • Ichki  style - bo`limdagi head element yordamida 
                    • Tashqi link -  tashqi CSS fayliga ulanish uchun element yordamida
                    CSS qo`shishning eng keng tarqalgan usuli bu uslublarni tashqi CSS fayllarida saqlashdir. Biroq, ushbu 
                qo`llanmada biz ichki uslublardan foydalanamiz, chunki buni ko`rsatish osonroq va uni o`zingiz sinab 
                ko`rishingiz     ham mumkin.
                           
Inline CSS 
  
                        Inline CSS yagona HTML elementiga noyob uslubni qo`llash uchun ishlatiladi.Inline CSS  style HTML 
                        elementning atributidan foydalanadi.quyidagi misol elementning 
                        matn rangini p  qizil rangga o`rnatadi: inlinecss.html        
                            
Ichki CSS

                Ichki CSS bitta HTML sahifasi uchun uslubni aniqlash uchun ishlatiladi.(head) ichki CSS HTML 
                sahifasining element ichida bo`limda aniqlanadi (style) . Quyidagi misol HAMMA elementning (h1) 
                (ushbu sahifadagi) matn rangini ko`k rangga , HAMMA elementning matn rangini esa (p) qizil rangga
                o`rnatadi. Bundan tashqari, sahifa "chang ko`k" fon rangi bilan ko`rsatiladi: ichkicss.html             
                            
Tashqi CSS

                        Ko`pgina HTML sahifalari uchun  uslubni aniqlash uchun tashqi uslublar jadvalidan foydalanadi.
                    (head) Tashqi uslublar jadvalidan foydalanish uchun har bir HTML sahifasi bo`limida unga havola qo`shiladi.
                    ya`ni quyidagi kabi : index.html  style.css
Eslatma! Tashqi uslular jadvali yordamida siz bitta faylni o`zgartirish orqali butun veb-sayt ko`rinishini o`zgartirishingiz mumkin!

CSS chegarasi

                CSS border xususiyati HTML elementi atrofidagi chegarani belgilaydi.
                Maslahat:Deyarli barcha HTML elementlari uchun chegara belgilashingiz mumkin.CSS_chegarasi.html      
            
p {
        border:  2px solid powderblue;
         }
